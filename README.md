# 버킷플레이스 Web Simple Copy

### 데모 사이트
https://bucketplace-web-simple-copy.web.app/

<br />

### 프로젝트 명령어
```
npm install : 프로젝트를 설치합니다.
npm start (또는 npm run dev) : 로컬 서버를 실행합니다.
npm run build : 배포 파일을 생성합니다.
``` 

<br />
