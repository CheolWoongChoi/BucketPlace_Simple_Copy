# 버킷플레이스 2020 웹 프론트엔드 과제

사진 피드를 만드는 과제입니다.

### 1. 프로젝트 명령어
```
npm install - 프로젝트 설치
npm start (또는 npm run dev) - 로컬 서버 실행
npm run build - 배포 파일 생성
```

### 2. 개발 스택
 - React
 - Redux
 - Redux-thunk
 - notistack (Material-UI 기반 Snackbar)
	 https://github.com/iamhosseindhv/notistack
 - lodash
 - axios
